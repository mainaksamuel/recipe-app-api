# recipe-app-api
Recipe App API source code

## Getting Started

Clone and go to the project directory

```bash
  git clone https://github.com/mainaksamuel/recipe-app-api.git
  cd recipe-app-api
```
To start the project, run:

```bash
  docker-compose up
```
The API will then be available at http://127.0.0.1:8000
